HƯỚNG DẪN TẠO BACKUP:
---------------------
1. Chạy script Python: python create_backup.py
   (Sẽ tạo file BACKUP_FULL_CODE_YYYYMMDD_HHMMSS.json)

2. Hoặc copy thủ công 3 file sau vào một thư mục backup:
   - index.html
   - assets/style.css
   - assets/script.js

HƯỚNG DẪN KHÔI PHỤC:
-------------------
1. Chạy script Python: python restore_backup.py BACKUP_FULL_CODE_YYYYMMDD_HHMMSS.json

2. Hoặc copy thủ công các file từ thư mục backup về vị trí gốc:
   - index.html -> index.html
   - style.css -> assets/style.css
   - script.js -> assets/script.js

CÁC FILE QUAN TRỌNG:
-------------------
- index.html: File HTML chính (823 dòng)
- assets/style.css: File CSS (5193 dòng)
- assets/script.js: File JavaScript (3766 dòng)

LƯU Ý:
------
- Lưu file backup ở nơi an toàn
- Kiểm tra file backup trước khi xóa code cũ
- Nên tạo backup trước mỗi lần chỉnh sửa lớn
